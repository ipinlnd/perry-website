The given brief for the first illustration : "Night time shot of the Ski Lodge Murder from a distance. Footsteps with blood leading to the House in an S curve." It is the game poster that will be used when released on Steam. Used sharp edges and strong shape design to contribute to the overall mood of a murder mystery game. Snow animation is done using Procreate on ipad.

The given brief for the second illustration : "B&W with toon shading and dark shadows in a purple hue. Shot from film noise overlay. Blown out exposure from the sky. Should allow for a character to be drawn in the centre / to the left third of the screen with design techniques drawing the viewer into the area. Could place a temporary character there for the composition guide." The second illustration will be used as a game asset. A chaotic riot in the city. The foreground character will be added later on.
